Installation

jsDeliver CDN

<script src="https://cdn.jsdelivr.net/npm/@thelevicole/toc.js@1.0.0/dist/toc.jquery.js"></script>

NPM / Webpack

npm i @thelevicole/toc.js
require('@thelevicole/toc.js');

Or

import '@thelevicole/toc.js'

Examples

  • Basic usage

HTML:

<div class="table-of-contents"></div>

<article>
	<h1>Heading 1</h1>
	<h2>Heading 2</h2>
	<h3>Heading 3</h3>
	<h2>Heading 2</h2>
	<h1>Heading 1</h1>
	<h1>Heading 1</h1>
</article>

Javascript:

(function($) {
	'use strict';

	$('.table-of-contents').tableOfContents({
		contentTarget: 'article'
	});

})(jQuery);

Result:

Heading 1

Heading 2

Heading 3

Heading 2

Heading 1

Heading 1
